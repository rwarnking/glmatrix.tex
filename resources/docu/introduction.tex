\section{Introduction}

The \textsf{glmatrix} package provides a small set of macros that allow to
do basic vector and matrix arithmetics in \LaTeX. It is inspired by the javascript
library with the same name providing the same functions also present in that library
and more. Using this library it is easy to do vector calculations like dot and cross
product as well as more complex calculations like vector matrix multiplication.

The package is specifically aiming for usecases in computer graphics,
consequently providing macros for vector transformations using matrices and
the creation of regularly used matrices in computer graphics like the lookAt matrix.
In combination these macros allow to do many regularly occuring calculations in Computer Graphics.
